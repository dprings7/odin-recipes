# odin-recipes
https://www.theodinproject.com/paths/foundations/courses/foundations/lessons/recipes

Create a basic website as per instructions that will contain 3 recipes

Learning objectives to demonstrate

- Create a website with multiple pages using the basic html boilerplate
- Link pages together using relative links
- Link to external sources using absolute links
- Include images using img tags, with alternate attributes
- Create an unordered list on each recipe page containing the ingredients
- Create an ordered list on each recipe containing the steps
